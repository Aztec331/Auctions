# 🛒 Auctions — Online Bidding Platform

## 🚀 Overview

Auctions is a full-stack web application that enables users to create listings, place bids, manage watchlists, and participate in an interactive online auction system.

The platform delivers a seamless bidding experience with real-time price updates, structured listing management, and user-specific activity tracking.

---

## ✨ Features

* 📝 Create and manage auction listings
* 💰 Place bids with automatic highest-bid tracking
* ⭐ Add and manage personal watchlist
* 🏷️ Browse listings by categories
* 💬 Comment on listings
* 🔒 User authentication and account management
* 📱 Responsive user interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Aztec331/Auctions.git
cd Auctions
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
source venv/bin/activate      # macOS / Linux
venv\Scripts\activate         # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run migrations

```bash
python manage.py migrate
```

### 5️⃣ Start the development server

```bash
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000
```
